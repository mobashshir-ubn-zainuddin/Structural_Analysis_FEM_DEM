FEM & DEM — Core Formulation Steps
From your professor's notes, extended with full mathematical formulations for 2D elements.
FEM Core Steps
  1. Discretization (Meshing)
    Divide geometry into triangular or quadrilateral elements connected at nodes.
  2. Shape Functions
    Interpolation functions N(x,y) relating field variable to nodal values.
  3. Element Stiffness Matrix
    ke = ∫ Bᵀ D B dV — using strain-displacement and constitutive matrices.
  4. Global Assembly
    Assemble Kglobal from all element matrices by node connectivity.
  5. Boundary Conditions
    Apply essential (Dirichlet) and natural (Neumann) BCs.
  6. Solve System
    K·u = F → u (nodal displacements) via Gaussian elimination.
  7. Post-processing
    Compute strains ε = B·u, stresses σ = D·ε.
DEM Core Steps
  1. Particle Initialization
    Set positions, radii, mass, velocity = 0, material properties.
  2. Contact Detection
    Check distance d < ri + rj for all particle pairs.
  3. Force Calculation
    Linear spring: Fn = kn·δ — Dashpot: Fd = γ·vn.
  4. Body Forces
    Apply gravity: Fg = m·g in −y direction.
  5. Time Integration
    Euler-Cromer (symplectic): v += (F/m)·dt, x += v·dt.
  6. Boundary Enforcement
    Wall contact forces with spring-dashpot model.
  7. Visualization
    Plot particle positions, contact networks, force chains.

CST — Constant Strain Triangle
3-node linear triangular element for 2D plane stress / plane strain problems.
Element Geometry
CST element with 3 nodes: (x₁,y₁), (x₂,y₂), (x₃,y₃). Each node has DOFs u (x-disp) and v (y-disp).
Shape Functions (Area Coordinates)
Natural / Barycentric coordinates
N₁ = (a₁ + b₁x + c₁y) / (2A) N₂ = (a₂ + b₂x + c₂y) / (2A) N₃ = (a₃ + b₃x + c₃y) / (2A) Where: a₁ = x₂y₃ - x₃y₂ b₁ = y₂ - y₃ c₁ = x₃ - x₂ (cyclic permutations for a₂,b₂,c₂,a₃,b₃,c₃) 2A = det|1 x₁ y₁| |1 x₂ y₂| |1 x₃ y₃|
Partition of unity condition:
N₁ + N₂ + N₃ = 1 (always)
Displacement Field

The displacement at any interior point (x,y) interpolated from nodal values:

u(x,y) = N₁·u₁ + N₂·u₂ + N₃·u₃ v(x,y) = N₁·v₁ + N₂·v₂ + N₃·v₃ In matrix form: {u} [N₁ 0 N₂ 0 N₃ 0 ] {u₁} {v} = [ 0 N₁ 0 N₂ 0 N₃] {v₁} {u₂} {v₂} → {d} = [N]{δₑ} {u₃} {v₃} DOF vector: {δₑ} = [u₁ v₁ u₂ v₂ u₃ v₃]ᵀ
Strain-Displacement Matrix [B]

For plane stress/strain, 3 strain components:

{ε} = [εₓ εᵧ γₓᵧ]ᵀ = [B]{δₑ} [B] = 1/(2A) · [b₁ 0 b₂ 0 b₃ 0 ] [ 0 c₁ 0 c₂ 0 c₃ ] [c₁ b₁ c₂ b₂ c₃ b₃ ] Note: B is CONSTANT → Constant Strain Triangle! Strains are uniform throughout the element.
Constitutive Matrix [D]
Plane Stress (thin plate, σz=0):
[D] = E/(1-ν²) · [1 ν 0 ] [ν 1 0 ] [0 0 (1-ν)/2 ]
Plane Strain (thick body, εz=0):
[D] = E/((1+ν)(1-2ν)) · [1-ν ν 0 ] [ ν 1-ν 0 ] [ 0 0 (1-2ν)/2 ]
Element Stiffness Matrix [kₑ]
kₑ = ∫∫ [B]ᵀ [D] [B] dA · t (where t = thickness) For CST, B is constant, so integration is trivial: kₑ = [B]ᵀ [D] [B] · A · t This is a 6×6 symmetric matrix: u₁ v₁ u₂ v₂ u₃ v₃ u₁ [k₁₁ k₁₂ k₁₃ k₁₄ k₁₅ k₁₆] v₁ [k₂₁ k₂₂ k₂₃ k₂₄ k₂₅ k₂₆] u₂ [k₃₁ k₃₂ k₃₃ k₃₄ k₃₅ k₃₆] v₂ [k₄₁ k₄₂ k₄₃ k₄₄ k₄₅ k₄₆] u₃ [k₅₁ k₅₂ k₅₃ k₅₄ k₅₅ k₅₆] v₃ [k₆₁ k₆₂ k₆₃ k₆₄ k₆₅ k₆₆] kᵢⱼ entries computed as: kᵢⱼ = Σ BᵢₖDₖₗBⱼₗ · A · t
Stress Recovery (Post-processing)
After solving: {σ} = [D][B]{δₑ} {σ} = [σₓ σᵧ τₓᵧ]ᵀ — same everywhere in CST element! Von Mises stress: σ_vm = √(σₓ² - σₓσᵧ + σᵧ² + 3τₓᵧ²) Principal stresses: σ₁,₂ = (σₓ+σᵧ)/2 ± √((σₓ-σᵧ)²/4 + τₓᵧ²)
Bilinear Quadrilateral Element (Q4)
4-node isoparametric element using natural coordinates (ξ, η) ∈ [−1, 1].
Isoparametric Mapping
Parent element in (ξ,η) space mapped to physical (x,y) space via Jacobian.
Shape Functions in Natural Coordinates
Defined over ξ ∈ [-1,1], η ∈ [-1,1]: N₁(ξ,η) = ¼(1-ξ)(1-η) → node 1 (-1,-1) N₂(ξ,η) = ¼(1+ξ)(1-η) → node 2 (+1,-1) N₃(ξ,η) = ¼(1+ξ)(1+η) → node 3 (+1,+1) N₄(ξ,η) = ¼(1-ξ)(1+η) → node 4 (-1,+1) Isoparametric concept: x = Σ Nᵢ·xᵢ (same functions for geometry) y = Σ Nᵢ·yᵢ u = Σ Nᵢ·uᵢ (and for displacements) v = Σ Nᵢ·vᵢ
Jacobian Matrix — Coordinate Transformation

Maps derivatives from natural to physical space. Critical for non-rectangular elements.

[J] = [∂x/∂ξ ∂y/∂ξ] = Σᵢ [∂Nᵢ/∂ξ · xᵢ ∂Nᵢ/∂ξ · yᵢ] [∂x/∂η ∂y/∂η] [∂Nᵢ/∂η · xᵢ ∂Nᵢ/∂η · yᵢ] det(J) must be > 0 everywhere (element validity condition) Inverse mapping (for B matrix computation): [∂Nᵢ/∂x] [J]⁻¹ [∂Nᵢ/∂ξ] [∂Nᵢ/∂y] = [∂Nᵢ/∂η]
Strain-Displacement Matrix [B] for Q4
[B] is 3×8 matrix (varies with ξ,η): u₁ v₁ u₂ v₂ u₃ v₃ u₄ v₄ εₓ: [∂N₁/∂x 0 ∂N₂/∂x 0 ...] εᵧ: [ 0 ∂N₁/∂y 0 ∂N₂/∂y ...] γ: [∂N₁/∂y ∂N₁/∂x ∂N₂/∂y ∂N₂/∂x...] Where ∂Nᵢ/∂x = J⁻¹[1,1]·∂Nᵢ/∂ξ + J⁻¹[1,2]·∂Nᵢ/∂η
Stiffness Matrix by Gauss Quadrature
kₑ = ∫₋₁¹∫₋₁¹ [B]ᵀ[D][B] |J| t dξ dη Numerical integration (2×2 Gauss rule): kₑ ≈ Σᵢ Σⱼ wᵢwⱼ · [B(ξᵢ,ηⱼ)]ᵀ[D][B(ξᵢ,ηⱼ)]·|J|·t Gauss points & weights (2×2): Point ξ η weight 1 -1/√3 -1/√3 1.0 2 +1/√3 -1/√3 1.0 3 +1/√3 +1/√3 1.0 4 -1/√3 +1/√3 1.0
Full Q4 Stiffness Matrix Structure (8×8)
u₁ v₁ u₂ v₂ u₃ v₃ u₄ v₄ u₁ [K₁₁ K₁₂ K₁₃ K₁₄ K₁₅ K₁₆ K₁₇ K₁₈] v₁ [K₂₁ K₂₂ K₂₃ K₂₄ K₂₅ K₂₆ K₂₇ K₂₈] u₂ [K₃₁ K₃₂ K₃₃ K₃₄ K₃₅ K₃₆ K₃₇ K₃₈] v₂ [K₄₁ K₄₂ K₄₃ K₄₄ K₄₅ K₄₆ K₄₇ K₄₈] u₃ [K₅₁ K₅₂ K₅₃ K₅₄ K₅₅ K₅₆ K₅₇ K₅₈] v₃ [K₆₁ K₆₂ K₆₃ K₆₄ K₆₅ K₆₆ K₆₇ K₆₈] u₄ [K₇₁ K₇₂ K₇₃ K₇₄ K₇₅ K₇₆ K₇₇ K₇₈] v₄ [K₈₁ K₈₂ K₈₃ K₈₄ K₈₅ K₈₆ K₈₇ K₈₈] Symmetric: Kᵢⱼ = Kⱼᵢ | Kᵢⱼ = Σ_gp wgp · Bᵢ(ξ,η)ᵀ D Bⱼ(ξ,η) |J| t
Global Assembly & Boundary Conditions
How element matrices combine into the global system, and how BCs are applied.
Global Stiffness Matrix Assembly
Algorithm: 1. Initialize: [K_global] = 0 (size: n_dof × n_dof) 2. For each element e: a. Compute [kₑ] (local stiffness, size: ndof_e × ndof_e) b. Identify global DOF indices for each local DOF (connectivity table: element → global node numbers) c. Add: K_global[global_i, global_j] += kₑ[local_i, local_j] (for all i,j in element DOF pairs) For 2D problems with n nodes: - Total DOFs = 2n (u and v at each node) - K_global is (2n × 2n) symmetric, sparse, positive semi-definite Assembly in Python (using numpy ix_): K[np.ix_(global_dofs, global_dofs)] += k_local
Essential BCs (Displacement)
Method 1 — Row/Column Elimination: Strike the row and column of the fixed DOF. Move contributions to RHS {F}. Method 2 — Large Number (Penalty): K[i,i] += α (α >> max(K entries)) F[i] = α · ū (ū = prescribed disp.) Method 3 — Lagrange Multipliers: [K cᵀ][u] = [F] [c 0 ][λ] [g] (adds λ = constraint force as unknowns)
Natural BCs (Traction / Load)
Distributed surface traction on edge: fₑ = ∫_Γ [N]ᵀ {t} dΓ For CST edge (nodes i,j, traction tₓ,tᵧ): fᵢₓ = (tₓ · Lᵢⱼ · t) / 2 fᵢᵧ = (tᵧ · Lᵢⱼ · t) / 2 fⱼₓ = fᵢₓ (equal sharing for linear elem.) fⱼᵧ = fᵢᵧ Body force vector: fᵦ = ∫∫ [N]ᵀ {b} dA (e.g. gravity {0, -ρg})
Global System and Solution
Global FEM system: [K_global] {U} = {F} Where: [K] = n_dof × n_dof symmetric positive definite (after BCs) {U} = vector of ALL nodal DOFs (unknowns) {F} = external force vector (applied + reactions) Partition into free (f) and constrained (c) DOFs: [Kff Kfc][Uf] [Ff] [Kcf Kcc][Uc] = [Fc] Free DOFs (what we solve for): [Kff]{Uf} = {Ff} - [Kfc]{Uc} Solution methods: • Direct: Gaussian elimination, LU decomposition, Cholesky • Iterative: Conjugate Gradient (large sparse systems) In Python: U_free = np.linalg.solve(K_reduced, F_reduced)
Complete Python Workflow — 2D CST Example
import numpy as np def cst_stiffness(coords, E, nu, t, mode='plane_stress'): x1,y1, x2,y2, x3,y3 = coords.flatten() A = 0.5*abs((x2-x1)*(y3-y1) - (x3-x1)*(y2-y1)) b1=y2-y3; b2=y3-y1; b3=y1-y2 c1=x3-x2; c2=x1-x3; c3=x2-x1 B = (1/(2*A)) * np.array([ [b1, 0, b2, 0, b3, 0], [ 0, c1, 0, c2, 0, c3], [c1, b1, c2, b2, c3, b3]]) if mode == 'plane_stress': D = (E/(1-nu**2))*np.array( [[1,nu,0],[nu,1,0],[0,0,(1-nu)/2]]) else: # plane_strain D = (E/((1+nu)*(1-2*nu)))*np.array( [[1-nu,nu,0],[nu,1-nu,0],[0,0,(1-2*nu)/2]]) k = B.T @ D @ B * A * t return k, B, D # Global assembly K = np.zeros((n_dof, n_dof)) for el in elements: dofs = get_dofs(el) # [u1,v1,u2,v2,...] indices k_e,_,_ = cst_stiffness(node_coords[el], E, nu, t) K[np.ix_(dofs, dofs)] += k_e # Apply BCs and solve K_red = K[np.ix_(free_dofs, free_dofs)] F_red = F[free_dofs] - K[np.ix_(free_dofs, fixed_dofs)] @ u_fixed U_free = np.linalg.solve(K_red, F_red)
DEM — Distinct Element Method
Complete formulation with contact mechanics, force models, and time integration schemes.
DEM Governing Physics
Newton's 2nd Law for each particle i: Translational: m_i · ẍᵢ = Σⱼ Fᵢⱼᶜ + Fᵢᵍ + Fᵢᵉˣᵗ Rotational: I_i · θ̈ᵢ = Σⱼ Mᵢⱼ Where: m_i = ⁴⁄₃ π r³ ρ (sphere mass) I_i = ²⁄₅ m_i r² (moment of inertia) Fᵢⱼᶜ = contact force from particle j on i Fᵢᵍ = gravity = m·g (downward) Mᵢⱼ = torque from tangential contact forces
Contact Detection
Step 1: Check overlap for pair (i,j): dᵢⱼ = |xⱼ - xᵢ| (distance between centers) δ = rᵢ + rⱼ - dᵢⱼ (overlap / penetration) Contact exists if δ > 0 Unit normal vector (i→j): n̂ᵢⱼ = (xⱼ - xᵢ) / dᵢⱼ Normal relative velocity: vₙ = (vⱼ - vᵢ) · n̂ᵢⱼ
Contact Force Models
1. Linear Spring-Dashpot (LSD): Fₙ = kₙ·δ - γₙ·vₙ (normal) Fₜ = -kₜ·δₜ - γₜ·vₜ (tangential) 2. Hertz-Mindlin (nonlinear): Fₙ = (4/3)E*√(R*)·δ^(3/2) E* = combined Young's modulus R* = rᵢrⱼ/(rᵢ+rⱼ) (effective radius) 3. Coulomb friction limit: |Fₜ| ≤ μ·|Fₙ| (sliding condition) If exceeded: Fₜ = μ·|Fₙ|·sign(vₜ)
Time Integration Schemes
1. Euler-Cromer (Symplectic Euler) — from your code: vᵢ(t+dt) = vᵢ(t) + aᵢ(t)·dt ← update velocity first xᵢ(t+dt) = xᵢ(t) + vᵢ(t+dt)·dt ← use NEW velocity Better energy conservation than standard Euler. 2. Velocity Verlet (most common in DEM): xᵢ(t+dt) = xᵢ(t) + vᵢ(t)·dt + ½aᵢ(t)·dt² aᵢ(t+dt) = Fᵢ(t+dt) / mᵢ ← recalculate forces vᵢ(t+dt) = vᵢ(t) + ½[aᵢ(t)+aᵢ(t+dt)]·dt 3. Critical time step (stability): dt_cr = 2/ωmax ≈ 2√(m_min/k_max) ← must not exceed this! Use dt ≈ 0.2 · dt_cr for safety.
Wall Boundary Conditions
For a wall at y = 0 (bottom boundary): δ_wall = rᵢ - yᵢ (overlap with wall) if δ_wall > 0: Fwall_y = kₙ·δ_wall - γₙ·vᵢᵧ (repulsive spring-dashpot) forces[i, 1] += Fwall_y For side walls at x = 0 and x = L: Left: if xᵢ < rᵢ → F_x = +kₙ(rᵢ-xᵢ) - γₙ·vᵢₓ Right: if xᵢ > L-rᵢ → F_x = -kₙ(xᵢ-(L-rᵢ)) - γₙ·vᵢₓ Alternative (simpler, from your code): np.clip positions to keep particles inside domain — but loses momentum continuity.
